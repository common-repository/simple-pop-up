== Description ==
This code was similar to w3school example:
https://www.w3schools.com/howto/tryit.asp?filename=tryhow_css_modal

For people who want a simple modal. I ported the w3school example here.

How to use : 

1. Get the id of the modal you want.

2. Enter the name of the button.

[pop_me_up id='id of pop up' btn_name="name of button"]

== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload `simple-message-modal.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `&lt;?php do_action('plugin_name_hook'); ?&gt;` in your templates
